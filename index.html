Create And Build With Your VOICE.


Ava is productive tool to automate applications and websites on the go.


Why Ava?


Ava is an easy to use application to automate desktop
tasks or websites on a system. Plus it's free to use.


How to use?


Just create a task template and do the task that you wish to automate and let Ava handle the rest. When finished you will have an automated task created for you that can be run anytime.
